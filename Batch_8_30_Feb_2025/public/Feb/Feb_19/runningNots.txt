HTML Attributes :
    Adding extra / meta data to tags through html Attributes

    <abc>
    </abc>

    Predefined Attributes:
        id -> using which we could add unique ref to html elements
        name -> we can group relative set of items with same name 
        class -> using which we can inject single or multiple classes to html elements
        style -> using which we can inject inline css prop 
        src - through which we can specify the path
        href - through which we can specify the path of external link
        alt - alternative text
        title - title to be shown on mouseover of element
        etc. 

    Syntax:
        <tagname attrname="value" attrname="value2" ... ... ..>
            ....
        </tagname>

        <human aadhar="2423423" contact="112312" name="asdfa" age="70">
            ...
        </human>

    Naming Convention: 

        <p id="newsContent_3"></p>

            id ="news cntent 1" - wrong 

            id ="newscontent1" -> works not recommended

            id="newsDetailsContent1" - works and recommended

            id="news-dtails-content1" - works
